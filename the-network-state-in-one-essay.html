    • 一文总结网络国家

    光是提设想, 算不得建国. 在这里我们会详尽描述一个和平的, 可复制的工作流, 能将一个有共同愿景的线上社区逐步转化为一个有着实体土地和虚拟首都网络国家 - 现行民族国家的新阶段进化版.

    网络国家公民不屑于隐瞒自己的观点和意图: 我们想要创建全新的国家的目的, 和我们想要一片处女地, 一张白纸, 一个未初始化的文本缓冲区, 一个初始的创业公司的目的是一样的. 我们想要抛开历史包袱, 在纯净全新的状态下创造!

    从财务层面来看, 需要全新环境的原因是非常清晰的. 人们每年购买几百万英亩的空白地块, 注册几十万家新公司实体, 全为了获得这一纯净状态. 如今, 不止是注册新的公司实体, 我们已经又获得了发起新的社区, 甚至发行新货币的能力. 人们已经对这些全新的创造方式趋之若鹜, 甘之如饴了.

    从社会层面来看, 全新环境的价值也是清晰的. 过去几十年间, 单单是在技术行业, 创建新公司的自由就已经在社会上创造了万亿级别的公司市值. 想象我们如果生活在这样一个世界, 如果要在纸上写东西, 就必须先把纸上之前的东西 擦掉, 要盖楼就必须要先拆掉一栋旧楼, 不许注册新公司只能重组一家壳公司, 那么纸张, 土地, 公司都会成为稀缺资源, 纠纷和内耗将是巨大的.

    等等, 好像都不需要想象 - 这不就是我们当下生活着的世界? 远古时代, 人们只能在泥板上刻下永远, 不远的几十年前, 在一些国家创立私人公司私营经济可判处死刑, 而就在现在, 旧金山的人们也在为是否要取代掉一间古老的加油站而争执不休. 以上提到的情况中, 要么因为技术限制, 要么因为政治因素, 要么因为法律规定, 人们无法轻易复原全新的状态, 或者不被允许重新创造.

    这恰恰就是当下社会, 对于国家 / 城市 / 民族 / 政府 / 以及大部分实体世界建构的现状. 由于建立新国家是不可想象的, 人们困于已有的旧建构, 争斗不休.

    也许我们能改变这一切.

    如何建立新国家 

    有至少六种方式能够建立新国家. 有三种是更惯常的, 另三种是更离经叛道的. 我们会介绍所有这六个方式, 最终引出我们设想的第七种解决方案.

    1. 选举 

    最通常的建立新国家的方式是通过在选举中赢得多数选票支持. 之后通过修改旧国家中既有法律, 或从旧国家中基于国际社会的承认再建立新的国家. 这是一条被讨论最多, 也是最常用最拥挤的路径 - 也许已经太拥挤了.

    2. 革命 

    第二个显然的方式是政变和革命. 我们并不推荐轻易尝试革命. 虽然一些重大的选举和改革也同样被称为革命 (revolutions), 但我们所说的革命的核心属性是流血冲突. 革命并不常发生, 但成功的革命大多数时候意味着新政权的建立.

    3. 战争 

    第三个常见的建立新国家的方式是赢得一场战争. 我们也并不推荐尝试这个方式. 当然, 战争是政治的延续, 特定的一场选举和革命 都能导致战争的爆发, 并最终带来新的政治后果. 和革命一样, 战争是少见并且代价巨大的, 并也的确能够建立新国家和政权.

    4. 微国家 

    现在我们来到不常见的领域. 最显然的, 并且也是大多数人听到 "建立新国家" 这句话时立刻会想到的一种方式, 就是一个怪人来到某个年久失修的海上离岸平台 或争议中的弹丸之地, 并自顾自地竖起一面国旗, 宣誓主权. 如果说, 通过选举来建立新国家是一种牵扯太多人的方式, 那么对于这些微国家 来说问题就是太少的人在乎他们了. 由于一个国家 (就像货币一样) 天然地是一个社会建构, 一个网络效应产品, 显然不毛之地的一小撮人无法组织军队, 执行法律, 或是被其他国家承认. 更进一步说, 有些情况下一个现存国家也许能容忍人们在自己后院里跑团RPG式地建立一个无害的小国, 可一旦对统治或国土产生了真正的威胁, 暴力镇压会立刻到来. 在马岛或是库页岛都是如此.

    5. 海上家园 

    这里事情开始变得有趣了. 由Patri Friedman和另一些作者提出, 并被Peter Thiel认可的海上家园概念, 是从一个简单的观察开始的: 游轮在公海上能持续几周的时间, 能装载几千人的乘客, 如果让游轮和乘客都半永久地在不受任一单个国家法律限制的公海上定居 (当然, 可以常常靠港), 会如何呢? 如果游轮的成本未来进一步下降, 这个方案还会变得更加可行. 唯一的问题是, 虽然我们看到过在海上生活多年的一些人的例子, 游轮式海上家园如何规模化扩张有待回答.

    6. 太空 

    建立新国家的最高贵酷炫的方式, 可能就是太空殖民了. 与海上家园或微国家不同, 太空探索由国家政府直接发起, 并在大众媒体和文艺作品中被渲染美化. 因此, 太空殖民有着相对更高的社会接受度. 这个路线被认为是技术上暂不可行的, 而不是 (像某几个上面的方式) 被认为是纯粹的疯子. Elon Musk的 SpaceX是其中一家认真在思考如何 在火星上建国的组织.

    7. 网络国家 

    欢迎. 我们终于来到本书相信的建立新国家方式: 网络国家. 我们的思路是「云端先行, 最终土地」. 比起首先考虑占领土地, 我们将先发起一个数字社区. 我们将创建一个初创社会, 将它组织为一个 网络公会, 然后众筹募资来购买我们的网络群岛的一个个物理节点 (地块). 经过时间的酝酿和积累, 在时机成熟之际最终得到国际社会的承认, 成为真正的网络国家. 我们将像开源项目一样培养萌芽中的国家, 用远程工作来组织国家经济, 培养与现实同样的公民礼仪, 在VR中搭建建筑, 创作符合我们社会价值观的文学与艺术作品.

    当我们在现实世界中众筹领土时, 它不一定是连续的土地. 因为一个被低估的事实是互联网允许我们建立网络飞地. 换句话说, 一个网络群岛不需要一次性地在一个集中的地方获得其所有领土. 它可以将不同城市的一千间公寓, 一百栋房屋和十几个死胡同连接成一种新的分形政体, 再加上一个 (Discord, Telegram...) 云端首都. 社区成员在这些飞地和附近的众筹到的领土之间迁徙, 由于这样的飞地领土有一千块, 每个单独的住宅和集体住宅都提供了独立的扩张机会, 所以扩张领土不是向四个方向 (东 / 南 / 西 / 北), 而是四千个方向.

    到目前为止, 我们所描述的很像一个流散移居的民族. 其中侨民分散在全世界, 但通过沟通渠道与彼此和祖国联系在一起. 不同之处在于, 我们的版本是一个反向流散: 一个社区在互联网上首先形成, 在线上建立一种文化, 然后才亲自聚集在一起建造定居点和其他社会结构. 某种意义上说, 可以将这个数字社区的实体前哨站都视为一个个云大使馆, 类似于世界各地兴起的帮助人们更好地了解比特币的草根比特币大使馆. 新的潜在成员可以访问网络国家的线上或实体部分, Beta测试试玩一番, 然后决定离开或留下.

    伴随所有这些关于大使馆和国家的讨论, 人们可能会开始嘲笑: 网络国家, 就像前面提到的微国家一样, 也只是一场RPG跑团, 一个LARP游戏. 然而与微国家不同的是, 它们被设计规划为一个能够扩大, 能够规模化的LARP. 一个由大量人共同想象, 勠力实践的壮举. 过去十年加密货币的经验向我们展示了这种大型多人LARP有多强大的力量.

    最小必要创新 

    让我们停下来总结一下第七种武器 - 网络国家 - 与前六种方式 (选举 / 革命 / 战争 / 微国家 / 海洋家园和太空) 的主要区别. 那就是, 第七种网络国家跨越了实践可行和不可行之间的鸿沟.

    现在人已经实际可行地可以建立百万人在线社区, 启动市值数十亿美元的数字货币, 在虚拟世界中建造建筑物, 然后众筹让它来到现实. 网络国家这一思想将许多现存技术堆叠在一起, 而不是需要发明全新的技术 - 比如火星运载飞船, 或永久性海洋基地. 同时, 它避免了选举, 革命和战争等方式显而易见的弊端 - 它们大多数会引致丑陋的争斗和惨痛的结局, 并且没有为个人努力留下什么空间.

    网络国家汇集我们拥有的最强大的技术栈 - 即互联网和互联网的支撑技术 - 之大成, 并且无需等待未来的实际技术创新, 可以绕过政治阻碍, 即刻启动.

    怎样才算一个国家? 

    在概述了这七种路径之后, 细心的读者会注意到, 我们还没有严谨地说清 "新国家" 的定义.

    首先, 我们所说的新国家到底是什么意思? 一个定义是, 一个新国家意味着必须有一个全新的领土, 例如火星. 另一个定义是, 只要改变政府形式就算换了一个国家, 就像法国从法兰西第二共和国转移到法兰西第二帝国一样. 我们将使用新国家的数值和社会定义, 而不是用前面这些或严格或松散的传统定义.

    首先, 想象有一个假想的网站 nationalrealestatepop.com 网站, 类似于 coinmarketcap.com. 该网站汇总了渴望成为网络国家的初创社会, 以及它们的经过密码学审计的人口普查数据. 该页面将实时显示社区成员的数量, 这些成员拥有的土地房地产面积以及社区的链上收入. 一个在全球拥有500万社区人口, 数千平方英里 (不连续) 社区土地以及数十亿年收入的初创社会将具有无可争辩的价值和重要性.

    这又将我们引向了社会定义: 一个新国家是一个在外交上被其他国家承认为能够自决的合法政体. 一个拥有足够多双边关系的国家将具有社会意义, 也有机会加入东盟 / 美洲国家组织 / 非洲联盟 / 欧盟或联合国等现有组织.

    这种数值和社会指标的组合与加密货币的发展轨迹是类似的: 最初被忽视, 然后被嘲弄为显然明天就要倒, 比特币被发明的五年后达到了十亿美元的市值 (数值上的进展来了!), 随后在CNBC和彭博上与其他蓝筹股票 / 大宗商品股一起显示价格 (这是社会认可上的进展和突破). 每一步, 比特币仍能保持数值上升, 随之而来的是更大的社会认可. 到 2020 年, 它已经影响了一系列银行和组织的看法和决策, 包括中国人民银行 / 国际货币基金组织, 高盛, 摩根大通和 世界银行. 到2021年, 比特币成为主权国家萨尔瓦多的法定货币. 到2022年年中, 中非共和国紧随其后认可了比特币为法定货币, 还有包括巴拿马在内的数十个国家正在考虑将比特币定为法定货币.

    大多数国家都是小国! 

    加密货币可以达到这样的高度, 是因为货币也是从数值和社会两个方面来衡量的. 在获得广泛社会接受之前, 加密货币的数值体量先涨了起来. 一旦比特币证明它极难被伪造或被破解, 全球数百万 加密货币持有者的共同信念就足以让比特币市值从零上升到数十亿, 并登上每个彭博终端和交易所. 这样的社会影响力增长, 又为为更大的数值增长铺平了道路, 随之而来的是一个良性循环.

    一个初创社会能否走上同样的道路? 完全可以. 可基于密码学审计的人口普查数据可以证明, 一个不断发展的初创社会确实拥有 1-1000 万忠实的数字公民, 大量的加密货币国库储备, 长年的持续存在, 以及遍布全球的实物资产. 这样的数值增长势头, 能用来推动外交承认的社会影响力增长.

    确实是如此吗? 是的, 因为大多数国家都是小国! 一个拥有 1-1000 万人口的新国家实际上可以与大多数现有国家相媲美. 这是因为联合国承认的 193 个主权国家中, 20% 的人口不到 100 万, 55% 的人口不到 1000 万. 这包括许多通常被认为是非常 "正规", 相当 "像样" 的国家, 例如: 卢森堡 (61.5万), 塞浦路斯 (120 万), 爱沙尼亚 (130 万), 新西兰 (470 万), 爱尔兰 (480 万) 和新加坡 (580 万) 按照技术公司标准，这些产品的用户量相当小!

    译者按: 另一些例子还包括 - 牙买加 270万, 卡塔尔 280万, 蒙古 340万, 挪威 540万, 芬兰 550万, 丹麦 590万 - 人口均少于萨尔瓦多.

    当然, 数量并不是一切. 我们设想中的网络国家与国民的紧密程度也很重要. 这包括在网络上花的时间, 以该网络国家货币形式存储的资产占比, 以及所有联系人里在这个网络国家中的比例也很重要.

    虽然有种种限定条件, 但一旦我们记住 Facebook 拥有 三十多亿 用户, Twitter 有 三亿多, 许多个人影响者拥有 100万 以上的追随者, 那么, 建立一个具有真正国家意识的 1-1000 万人初创社会, 一种集成在其中的加密货币, 以及一项众筹购买全球许多土地的计划就不再是 个太疯狂的主意. 借助互联网, 我们可以将这些不相连的飞地以数字方式缝合在一起, 形成一种能获得主权承认的新型政体: 网络国家.
